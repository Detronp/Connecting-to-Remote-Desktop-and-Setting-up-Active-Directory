
# Connecting-to-Remote-Desktop-and-Setting-up-Active-Directory
## Remote Desktop Connection Setup for Virtual Machine

### Objective

This SOP outlines the steps to connect to a virtual machine using Remote Desktop Protocol (RDP) and set up Active Directory.

### Key Steps

 

**Step 1: Access the Virtual Machine** [0:01](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=1)

![generated-image-at-00:00:01](https://loom.com/i/5e89777c06864243a9340d7f55927df8?workflows_screenshot=true)

- Ensure the virtual machine is running and initializing.
- Prepare to connect to the virtual machine using RDP.

 

**Step 2: Open RDP Client** [0:10](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=10)

![generated-image-at-00:00:10](https://loom.com/i/8fd76f596def4477a111b76d81ab47cd?workflows_screenshot=true)

- Locate and open the Remote Desktop Protocol (RDP) client on your computer.

 

**Step 3: Retrieve Password** [0:22](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=22)

![generated-image-at-00:00:22](https://loom.com/i/45d52c74516e43ae966d2491d93ba028?workflows_screenshot=true)

- Find the remote desktop file associated with the virtual machine.
- Decrypt the password from the file.

 

**Step 4: Connect to the Virtual Machine** [0:59](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=59)

![generated-image-at-00:00:59](https://loom.com/i/04fbd45683b3479cbd2fe2d973691b51?workflows_screenshot=true)

- In the RDP client, enter the IP address of the virtual machine.
- Paste the decrypted password into the password field.

 

**Step 5: Launch Remote Desktop Session** [1:10](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=70)

![generated-image-at-00:01:10](https://loom.com/i/ca532588f5e747b5b16862b78583376c?workflows_screenshot=true)

- Click 'Connect' to initiate the remote desktop session.

 

**Step 6: Open Server Manager** [1:20](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=80)

![generated-image-at-00:01:20](https://loom.com/i/9c8152e8f27548bc8908e32990407618?workflows_screenshot=true)

- Once connected, navigate to the Server Manager on the virtual machine.

 

**Step 7: Set Up Active Directory** [2:24](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=144)

![generated-image-at-00:02:24](https://loom.com/i/c5bc5a6e0a9b473392993d59ad96def6?workflows_screenshot=true)

- Begin the setup process for Active Directory while connected to the remote desktop.

 

**Step 8: Monitor Progress** [2:37](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=157)

![generated-image-at-00:02:37](https://loom.com/i/6c87a7496c86451eba5792f0f3b954ab?workflows_screenshot=true)

- Wait for the Server Manager to collect inventory and complete loading.

 

**Step 9: Follow Up** [2:46](https://loom.com/share/7ea2b17ab0aa4600a43f74fd7577f584?t=166)

![generated-image-at-00:02:46](https://loom.com/i/b9d74af010e44debb50730356a858d14?workflows_screenshot=true)

- If the process takes too long, note to return to complete the setup in the next session.

### Cautionary Notes

- Ensure that the virtual machine is fully initialized before attempting to connect.
- Keep the decrypted password secure and do not share it with unauthorized personnel.

### Tips for Efficiency

- Use a password manager to store and retrieve passwords securely.
- Familiarize yourself with the RDP client settings to streamline future connections.
